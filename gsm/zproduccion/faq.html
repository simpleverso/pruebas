Preguntas frecuentes

Aquí encontrarás las respuestas a las preguntas más comunes sobre Didxazapp, su traductor experimental y la API.

¿Las traducciones son fiables?

Tenemos buenos resultados, pero ésta es una aplicación experimental y no es recomendable su uso para traducción de documentos oficiales, información médica, canciones, libros, etc. Para esos casos recomendamos servicios de traducción humana.

¿Reciben apoyo de instituciones públicas?

No. Este proyecto no recibe apoyo de ninguna institución pública, gubernamental, educativa, ni de ninguna entidad relacionada con el cuidado de las lenguas indígenas en México.

¿La app requiere internet?

Sí. Las traducciones desde la app gratuita se obtienen de un servicio en línea y requieren internet para enviar y recibir información. La traducción por mensajes de texto (SMS) no requiere internet, pero sí saldo o un plan de SMS para su uso.

¿Cómo funciona Didxazapp?

Didxazapp genera números a partir de una frase en español, los cuales son procesados, analizados y traducidos mediante reglas matemáticas en un servicio en línea, que convierte esos resultados en texto comprensible, tal como lo hacen otros traductores como Google Translator o Bing Translator.

¿Se guardan mis datos personales al usar la app, el sitio o la API?

No. Ni la aplicación, ni el sitio en internet, ni la API recolectan información sobre datos personales.

¿La aplicación está respaldada por alguna autoridad de lenguas indígenas en México?

No. Éste es un proyecto totalmente independiente. No está involucrado ni recibe recursos, información o ayuda de ninguna dependencia.

¿De dónde vienen los datos que sirven como traducción?

Los datos se obtienen de las distintas fuentes disponibles en internet, libros, revistas y otras publicaciones, las cuales se analizan para identificar la información que puede ser agregada al sistema traductor.

¿Por qué hay un límite de 25 letras al traducir en la app?

El límite es un método para mejorar el rendimiento del servicio, de modo que los dispositivos con menos capacidades puedan aprovechar las funciones de traducción de la misma forma que los equipos de nueva generación.

¿Puedo agregar el servicio de traducción Didxazapp a mi aplicación?

Sí. Es necesario cumplir con algunos requisitos; una vez validada la información, te proporcionaremos las credenciales para usar el sistema de traducción directamente desde tu aplicación o desarrollo. Para más información: myred.zapoteca@gmail.com.

¿Cuentan con API de desarrollo?

Sí. Contamos con una API sencilla y funcional para Android, iOS, Windows Phone, Windows Desktop, HTTP, etc. Para más información: myred.zapoteca@gmail.com.
